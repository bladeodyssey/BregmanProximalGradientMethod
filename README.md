# Personal Homepage (Static Site)

这是一个简洁现代的个人主页静态网站项目，包含：
- 头像占位、姓名与简介
- 技能标签
- 3 个项目卡片
- 联系按钮（Email）
- 响应式布局与轻微交互动效

## 文件结构

- `index.html`：页面结构
- `styles.css`：页面样式与响应式布局
- `script.js`：轻量动效脚本

## 本地打开方式

### 方式 1：直接双击打开
1. 在文件管理器中找到 `index.html`
2. 双击即可在浏览器中预览页面

### 方式 2：使用 VS Code Live Server
1. 用 VS Code 打开项目目录
2. 安装扩展 **Live Server**（如果尚未安装）
3. 右键 `index.html`，选择 **Open with Live Server**
4. 浏览器会自动打开并支持刷新预览
